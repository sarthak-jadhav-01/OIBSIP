# Retail Sales Data Analysis (OIBSIP)

## Project Overview
This project analyzes retail sales data to find insights about customer behavior and sales trends.

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Analysis
- Data Loading
- Data Cleaning
- Product Category Analysis
- Monthly Sales Analysis
- Gender Analysis

## Key Insights
- Highest sales category: Electronics
- Peak sales month: May
- Female customers spend more

## Conclusion
Business should focus on Electronics and peak months to increase profit.
